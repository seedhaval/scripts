      -P

   ssh-keygen
      generate public and private pair of keys
   ssh-copy-id
      copy public key to the remote server

      securely copy files to a remote server
      -p
         use a non standard ssh port
   ~/.ssh
      directory that contains ssh related files
   authorized_keys
      details of host that are allowed to do ssh
