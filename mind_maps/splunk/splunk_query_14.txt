splunk query
    makemv
        convert single value field into multi value field
        delim
            delimiter to be used
            default is single space
        tokenizer
            regex capturing group. for each match, first capturing group is used as value
        makemv delim=',' senders
        makemv tokenizer="([^,]+),?" my_mv
    makeresults
        generate specified number of search results
        count=
            number of results to generate
            default is 1
        makeresults count=5
    multikv
        extract field values from table formatted search results
        example events generated by top, tstat
        requirement
            columns should be aligned
            the first line should contain names of data
        multikv fields pid command
            extract pid and command fields
