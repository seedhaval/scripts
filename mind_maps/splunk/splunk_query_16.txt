splunk query
    reltime
        create relative time fields
        human readable time
            1 minute ago, 3 hours ago
        difference between time field and now
        timefield=
            comma separated list of fields enclosed in double quotes
            if only 1 field provided, output field name is reltime
            default field _time
        prefix=
            prefix for new relative time fields
            default reltime_ when many fields, None when single field
            it does not explicity add _. you need to include it
        reltime
        reltime timefield=earliest_time
        reltime timefield="max_time,min_time" prefix="user_relative_"
    rename
        rename one or more fields
        wildcards can be specified
        rename sessionid as sid
        rename *ip as *ipaddress
        rename foo* as bar*
