splunk query
    mvexpand
        expand value of a multivalue field into separate events
        mvexpand foo
        only works on 1 multivalue field
    nomv
        convert multivalue into single value
        delimited by newline
        nomv senders
    rangemap
        categorize values in a numeric field
        adds a new field - range
        ranges can overlap. creates multivalue field if overlap
        rangemap field=date_second green=1-30 blue=31-39 red=40-59 default=gray
    rare
        display least common value in a field
        countfield=
            name of new field to write count into
        limit=
            how many tuples to return
            default 10
        rare url
        rare user by host
