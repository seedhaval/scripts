        subn( pattern, repl, string, count=0, flags=0 )
            same as sub but return (new_string, number_of_subs)

        Match
            group([group1,...])
                return subgroups of match
                without argument return whole match
                number or names can be given as arguments
            groups( default=None)
                return tuple containing all groups
                default= default value for non participating groups
            groupdict( default=None )
                dictionary of named groups
