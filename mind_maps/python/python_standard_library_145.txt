python standard library
    secrets
        secrets.randbits
            (k)
            getrandbits(k) -> x
            Generates an int with k random bits.
        secrets.token_bytes
            (nbytes=None)
            Return a random byte string containing *nbytes* bytes.
            If *nbytes* is ``None`` or not supplied, a reasonable default is used.
            >>> token_bytes(16)  #doctest:+SKIP
        secrets.token_hex
            (nbytes=None)
            Return a random text string, in hexadecimal.
            The string has *nbytes* random bytes, each byte converted to two hex digits
            If *nbytes* is ``None`` or not supplied, a reasonable default is used.
            >>> token_hex(16)  #doctest:+SKIP
        secrets.token_urlsafe
            (nbytes=None)
            Return a random URL-safe text string, in Base64 encoding.
            The string has *nbytes* random bytes
            If *nbytes* is ``None`` or not supplied, a reasonable default is used.
            >>> token_urlsafe(16)  #doctest:+SKIP
