python standard library
    built in functions
        abs
            get absolute number
        all
            return True if all elements are true or if iterable is empty
        any
            return True if any element in iterable is True
            for empty iterable, return False
        ascii
            return printable representation
            for non printable previx \x, \u in output
        bin
            convert integer to binary string
            prefix with 0b
        callable
            returns true if passed argument is callable
        chr
            unicode / ascii to character
        @classmethod
            transform a method into class method
            class is implicit first argument
            can be called either on class or instance
