    datetime
        timedelta( days, seconds, microseconds, milliseconds, weeks, hours, minutes )
            by default all arguments are 0
            represents duration. difference between 2 dates or times
            attributes
                days, seconds, microseconds
            total_seconds()
                number of seconds contained in the duration    
        date
            represents date - year, month, day
            date( year, month, day )
            today()
                current local date
