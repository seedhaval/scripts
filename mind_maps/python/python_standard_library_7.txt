python standard library
    built in functions
        repr
            printable representation of object
        reversed
            reverse iterator
        round( number[, ndigits] )
            if ndigits is not provided, round to integer
        set( [iterable] )
            convert iterable to set
        setattr( object, name, value )
            set name attribute of object to value
        slice( start, stop[, step] )
            a[start:stop:step]
        sorted( iterable, key, reverse=False )
            reverse = True - sort keys in reverse order
            key - function to be used to get sort key value
        @staticmethod
            does not receive an implicit first argument
            againt classmethod, where first argument is class
            can be called either on class or on instance
