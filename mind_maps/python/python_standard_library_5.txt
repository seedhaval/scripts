python standard library
    built in functions
        min
            return minimum from iterable
            default=
                return value when iterator is empty
            key=
                function to be used for sorting
        next( iterator, default )
            get next item from iterator
            if iterator exhausted, return default
        oct(x)
            convert integer to octal
        open
            open file
            mode
                r - read, w - write, a - append
                x - exclusive write, fail if file already exists
                b - binary, t - text
                + - reading + writing
                r+ - open without truncate
                w+ - open and truncate
