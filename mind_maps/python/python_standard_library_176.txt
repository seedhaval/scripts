        subprocess.call
            (*popenargs, timeout=None, **kwargs)
            Run command with arguments
            Wait for command to complete or timeout, then return the returncode attribute.
            The arguments are the same as for the Popen constructor
            Example:
            retcode = call(["ls", "-l"])
